# Traffic Light
A simple Arduino project to simulate a traffic light.

## Pre-requisites
- 7 x Jumper Wires
- 1 x Arduino Uno
- 1 x Breadboard
- 1 x 5mm LED: Red
- 1 x 5mm LED: Green
- 1 x 5mm LED: Yellow
